=========================================================
Mental_arithmetic
Программа для обучения устному счёту
=========================================================

Описание
--------
Mental_arithmetic — настольное приложение, разработанное на Python с использованием
библиотеки PyQt6. Программа предназначена для тренировки навыков устного счёта,
развития скорости вычислений, концентрации внимания и математического мышления.

Точка входа
-----------
main.py

Требования
----------
- Python 3.14 или выше
- Установленные зависимости из файла requirements.txt

Используемые библиотеки
-----------------------
Стандартные библиотеки Python:
- random
- json
- hashlib
- os
- datetime

Сторонние библиотеки:
- PyQt6
- PyQt6_sip

Структура проекта
-----------------
main.py          - точка входа в программу
data/            - файлы данных и конфигурации приложения
img/             - изображения, иконки и графические ресурсы
requirements.txt - список необходимых зависимостей

Установка
---------
1. Установить Python версии 3.14 или выше.
2. Открыть терминал в папке проекта.
3. Установить зависимости командой:

   pip install -r requirements.txt

Запуск программы
----------------
После установки зависимостей выполните команду:

   python main.py

Примечания
----------
Для корректной работы программы необходимо сохранить исходную структуру
проекта и не удалять папки data и img, так как они содержат необходимые
данные и графические ресурсы приложения.
